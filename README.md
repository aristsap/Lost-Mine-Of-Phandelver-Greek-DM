# Lost Mine of Phandelver (Ελληνική Προσαρμογή για Dungeon Masters)

Αυτό το αποθετήριο περιέχει μια πλήρη οργάνωση και μετάφραση στα ελληνικά της κλασικής περιπέτειας **Lost Mine of Phandelver** για το Dungeons & Dragons 5e. 

Σκοπός του είναι να βοηθήσει τους Έλληνες Dungeon Masters (ειδικά τους νέους) να τρέξουν την περιπέτεια πιο εύκολα, έχοντας όλες τις σημειώσεις, την πλοκή και τους χαρακτήρες μεταφρασμένα και δομημένα για γρήγορη αναφορά κατά τη διάρκεια του παιχνιδιού.

## Τι περιλαμβάνεται:

*   **Οργανωμένα Κεφάλαια:** Από την πρώτη ενέδρα μέχρι το επικό φινάλε στο Σπήλαιο της Ηχούς.
*   **Οδηγός για τον DM:** Βασικές συμβουλές και κανόνες για το πώς να προετοιμαστείτε και να τρέξετε το session.
*   **Πλήρες Background:** Η ιστορία της περιοχής και η σύνοψη της πλοκής.
*   **Φάκελος NPCs:** Αναλυτικά προφίλ για τους σημαντικότερους χαρακτήρες (κίνητρα, γνώσεις, τρόπος παιξίματος).
